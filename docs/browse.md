### GitHub Copilot (Student)

**What it is:** AI code suggestions and chat inside your IDE (GitHub).

**Keywords:** `software suite` · `coding development`

| Field | Value |
|-------|-------|
| **Listing status** | `active` |
| **Last checked** | 2026-05-17 |
| **Who qualifies** | Verified students on GitHub Education; at least 13 years old; personal GitHub account. |

> **What you get:** Free GitHub Copilot Student for verified GitHub Education students (unlimited completions; 300 premium requests/month per plan docs).

**Steps to apply**

1. Apply for GitHub Education as a student (degree/diploma program, age 13+, school email or proof).
2. Go to https://github.com/settings/education/benefits
3. Under "Free GitHub developer resources", open Copilot Student and follow activation prompts.
4. Alternatively claim via GitHub Student Developer Pack at https://education.github.com/pack

**[→ Verify on official site (docs.github.com)](https://docs.github.com/en/copilot/about-github-copilot/plans-for-github-copilot)**

Product homepage: [github.com](https://github.com/features/copilot)

*Region / eligibility notes:* GitHub Education student eligibility is global per apply-to-education docs; Copilot feature availability may vary by region.

### GitHub Student Developer Pack

**What it is:** Bundle of free dev tools, domains, and cloud credits for students.

**Keywords:** `student program` · `cloud api credits`

| Field | Value |
|-------|-------|
| **Listing status** | `active` |
| **Last checked** | 2026-05-17 |
| **Who qualifies** | Verified GitHub Education students (degree/diploma program; age 13+ per GitHub Education rules). |

> **What you get:** Free partner cloud and developer tools for verified GitHub Education students (e.g. $100 Azure credit, $200 DigitalOcean credit, GitHub Pro, Copilot Student).

**Steps to apply**

1. Apply for GitHub Education as a student at https://github.com/settings/education/benefits
2. Provide proof of enrollment (school email, student ID, schedule, transcript, or letter per GitHub docs).
3. After approval, open https://education.github.com/pack and claim each partner offer.
4. Follow the partner-specific redemption link for cloud credits (Azure, DigitalOcean, etc.).

**[→ Verify on official site (education.github.com)](https://education.github.com/pack)**

Product homepage: [github.com](https://github.com)

*Region / eligibility notes:* GitHub Education student verification is available globally per GitHub documentation; individual partner offers may have regional limits. Applicants must be at least 13 years old and in a degree/diploma-granting program.

### JetBrains Student Pack

**What it is:** Free professional IDEs—IntelliJ, PyCharm, WebStorm, and others.

**Keywords:** `software suite` · `coding development`

| Field | Value |
|-------|-------|
| **Listing status** | `active` |
| **Last checked** | 2026-05-17 |
| **Who qualifies** | University email, ISIC/ITIC, or GitHub Student Developer Pack; accredited program ≥1 year; non-commercial educational use only. |

> **What you get:** Free JetBrains IDEs and Student Pack for eligible students; includes a limited AI trial in the IDE.

**Steps to apply**

1. Visit https://www.jetbrains.com/community/education/#students
2. Click "Request now" and verify with university email, ISIC/ITIC card, or GitHub Student Developer Pack.
3. Alternatively apply at https://www.jetbrains.com/shop/eform/students
4. Renew annually via the link in your JetBrains Account before expiration.

**[→ Verify on official site (jetbrains.com)](https://www.jetbrains.com/academy/student-pack/)**

Product homepage: [jetbrains.com](https://www.jetbrains.com)

*Region / eligibility notes:* Full-time student in a program lasting one year or more. JetBrains AI service restricted to allowed territories per JetBrains legal docs (Mainland China limitations for some AI tiers).
