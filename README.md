# Student AI Offers

Community-maintained YAML records of **documented** student and education offers for AI-related tools. Each entry links an official `source_url` and a `verified_at` date—[offers change](CONTRIBUTING.md#how-to-update-verified_at); always confirm eligibility and price on the vendor’s site before you sign up.

**Disclaimer:** This repository is for information only. It is not legal, tax, or financial advice, and we are not affiliated with the vendors listed. Discounts, eligibility, and terms are set by each company and can change without notice.
